# DERMA-AI

Kami menggunakan dataset HAM10000, yang merupakan kumpulan citra penyakit kulit manusia, untuk mengklasifikasikan berbagai macam lesi kulit, kondisi kulit sehat, serta citra non-kulit. Selain itu, data untuk penyakit kudis ditambahkan dari Roboflow Scabies Computer Vision Dataset. Data gabungan ini kemudian diproses untuk menghasilkan metadata yang berfungsi untuk memisahkan citra menjadi set Data Training dan Data Testing.

Untuk dataset kita ambil dari link berikut :

HAM10000 : https://www.kaggle.com/datasets/roscoekerby/hamaugmentedwithnormalskinimages?select=akiec

Scabies : https://universe.roboflow.com/scabies/scabies-ps3vx

Non-Skin : https://www.kaggle.com/datasets/thepbordin/indoor-object-detection

!Dataset tidak kita masukkan karena terlalu banyak
